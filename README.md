# 一、Design-pattern设计模式
    常用的设计模式
## 1、工厂模式(Factory mode)
    不需要关注API的源码，只需要关注API的方法会返回什么结果
## 2、单例模式(Singleton mode)
    当一个方法返回一个对象的实例的时候，更加希望这个对象只实例化一次，其它的均是拿到这个对象的引用值
## 3、适配器模式(Adapter mode)
    当无法直接使用一个对象的方法时，可以通过实例化这个对象的方法，通过这个实例对象的方法调用这个对象
